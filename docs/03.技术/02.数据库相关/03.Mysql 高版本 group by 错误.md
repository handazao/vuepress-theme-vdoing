## 1. 修改sql_mode(全局修改)
```shell
select @@GLOBAL.sql_mode;

set @@GLOBAL.sql_mode='STRICT_TRANS_TABLES,NO_ZERO_IN_DATE,NO_ZERO_DATE,ERROR_FOR_DIVISION_BY_ZERO,NO_ENGINE_SUBSTITUTION'
```

> 注意:
> 
> 全局修改：需要有权限，并且不会立即生效，对以后新建的会话生效（从全局继承的）
